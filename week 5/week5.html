About CSS selectors

We can style HTML elements using different techniques.

We can target some elements more precisely with two types of selectors.

  • using descendant selectors
    • as long they are located inside that element
    • that's how you write the CSS property: ancestor descendant{styling}
  • using children selectors
    • when the element is located only one level deeper
    • that's how you write the CSS property: ancestor>child{styling}
